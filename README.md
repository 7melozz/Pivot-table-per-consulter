## 📝 Descrição
Ferramenta para criar e manipular tabelas dinâmicas (pivot tables) por consultor, utilizando Pandas para análise de dados.

## 🚀 Quick Start
```bash
git clone https://github.com/7melozz/Pivot-table-per-consulter.git
cd Pivot-table-per-consulter
pip install -r requirements.txt
```

## 🛠️ Tecnologias
- Python 3.8+
- Pandas 1.0+
- NumPy 1.21+
- Jupyter Notebook
